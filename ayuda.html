  • ¿Qué es encriptar un texto?

     

    Es el proceso de codificar un mensaje o información de modo tal que solo los individuos autorizados sean capaces de acceder a esta, y aquellos que no estén autorizados no puedan hacerlo. Encriptar un texto implica transformar la información original de manera que sea ilegible o difícil de entender sin tener la clave correcta para descifrarla. La encriptación se usa para guardar de manera segura la confidenciabilidad de la información que es privada o importante para no compartir a todo el mundo.

  • ¿Cómo usar esta página?

     

    Posee 3 funciones:  

    1. Permitir ingresar un texto y encriptarlo.

     

    2. Desencriptar un texto.

     

    3. Copiar el texto y pasarlo a otro lado.

      Es decir, en la página Home puedes digitar un texto que necesites encriptar (apareceran algunas advertencias como no usar acentos, ni caracteres especiales en caso de que encuentre esto en el texto ingresado), luego darle Enter y al lado derecho aparecerá el texto encriptado, puedes darle copiar si lo requieres.

  • ¿Qué tipos de encriptación se usa aqui?

     

    Las "llaves" de encriptación que se utilizan son las siguientes:

    La letra "e" será "enter",

    La letra "i" será "imes",

    La letra "a" será "ai",

    La letra "o" será "ober",

    La letra "u" será "ufat".

    Requisitos:

    Debe funcionar solo con letras minúsculas, No deben ser utilizados letras con acentos ni caracteres especiales, Debe ser posible convertir una palabra para la versión encriptada también devolver una palabra encriptada para su versión original.
